# Project Notes for Codex

This project is a Chrome extension for fantasy baseball helper workflows.

Prefer practical, meaningful improvements over tiny overly cautious changes. Keep changes focused, but do not avoid useful cleanup when it clearly supports the requested work.

When making a meaningful code, UI, behavior, or manifest change, update the extension version number in `manifest.json`.

Use patch versions for small fixes and tweaks. Use minor versions for new features.

After making changes, briefly summarize what changed and mention the new version number. For documentation-only changes that do not affect the extension, say that no version bump was needed.
